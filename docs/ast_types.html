XDC parser AST types

CompileUnit
  • Import[] imports
  • Function[] functions
Import
  • string module_name
Function
  • string name
  • DataType return_type
  • VariableDeclaration[] parameters
  • CodeBlock body
VariableDeclaration
  • DataType type
  • string name
variable_definition_t
  • data_type_t type
  • string name
  • expression_t initializer
DataType abstract
  • ( BasicType | static_array_type_t | dynamic_array_type_t )
BasicType
  • ( boolean_t | int_t )
static_array_type_t
  • basic_type_t data_type
  • integer length
CodeBlock
  • Statment[] statments
Statment abstract
  • ( variable_definition_t | assigment_t | code_block_t | for_loop_t | if_else_t | function_call_t | Return )
if_else_t
  • if_statment_t[] if_else_blocks
  • code_block_t else_block
if_statment_t
  • expression_t condition
  • code_block_t body
assigment_t
  • variable_ref_t target
  • expression_t value
Expression abstract
  • ( unary_operation_t | binary_operation_t | Constant | variable_ref_t | function_call_t )
unary_operation_t
  • unary_operation_type_t operation
  • expression_t operand
binary_operation_t
  • binary_operation_type_t operation
  • expression_t left
  • expression_t right
Constant abstract
  • ( IntConstant | BoolConstant | FloatConstant )
IntConstant
  • integer value
BoolConstant
  • boolean value
Return
  • Expression return_value
variable_ref_t
  • variable_ref_type_t type
  • ( string | array_cell_ref_t)
for_loop_t
  • ? init
  • expression_t test
  • ? step
  • code_block_t body
function_call_t
  • string name
  • expression_t[] arguments
array_cell_ref_t
  • string name
  • integer index
