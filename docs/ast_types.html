XDC parser AST types

compile_unit_t
  • import_t[] imports
  • function_t[] functions
import_t
  • string module_name
function_t
  • string name
  • data_type_t return_type
  • variable_defintion_t[] arguments
  • code_block_t body
variable_definition_t
  • data_type_t type
  • string name
  • expression_t initializer
code_block_t
  • statment_t[] statment
statment_t
  • statment_type_t type
  • ( variable_definition_t | assigment_t | code_block_t | for_loop_t | if_else_t | function_call_t | return_t )
assigment_t
  • lvalue_t target
  • expression_t value
expression_t
  • expression_type_t type
  • ( unary_operation_t | binary_operation_t | constant_t | variable_t | function_call_t )
function_call_t
  • string name
  • expression_t[] arguments
lvalue_t
  • lvalue_type_t type
  • ( variable_lvalue_t | array_cell_lvalue_t )
variable_lvalue_t
  • string name
array_cell_lvalue_t
  • string name
  • integer index
